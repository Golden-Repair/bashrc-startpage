# Start-Page
Start page made after a linux terminal with functional commands for shortcuts and searches.

Commands supported:
Touch [name] [url] (create link in current dir)

echo [Sentence] 

mkdir [name]

rmdir [name]

rm [name]

ls

clear

cd

locate [query](duckduckgo search)
